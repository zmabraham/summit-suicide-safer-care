# 6th Annual Summit Impact Report

A static impact-report microsite for **The Wellness Institute's 6th Annual Summit on
Suicide-Safer Care in Clinical Practice** (March 10–11, 2026), presented in partnership
with the American Foundation for Suicide Prevention (AFSP).

Implemented from a [Claude Design](https://claude.ai/design) handoff bundle
(`impact-report-site/index.html`).

## Contents

- `index.html` — the complete, self-contained site (inline CSS + JS)
- `fonts/` — Founders Grotesk brand typeface
- `brand/` — Summit key-art shapes
- `logos/` — partner & organization logos
- `speakers/` — faculty portraits

## Local preview

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Deployment

Served as a static site via GitHub Pages from the repository root.
